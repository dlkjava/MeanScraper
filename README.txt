Welcome to the MeanScraper

Step 1:
To get started, open 2 dos prompts.

In first dos prompt, type the following to start the server:
npm install
node app

In second dos prompt, type the following to start the client:
cd public
npm install
npm run tsc:w


Step 2:
open browser and navigate here:
http://localhost:3000/
